A Javascript Database

LokiJS is a javascript database that can be used in multiple scenarios.

Javascript is an easy to learn, versatile language, so database development in javascript can be easy and very productive. You may not want to retire your MySQL db just yet but here are some situationss in which you may find LokiJS to be the ideal solution:

  • Mobile applications - especially HTML5 based (Cordova, Phonegap, etc.)
  • Node.js embedded datastore for small-to-medium apps
  • Embedded in desktop application with Node Webkit

Installation

LokiJS is available on npm and bower. Run:
npm install lokijs
or:
bower install lokijs
