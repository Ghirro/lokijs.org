A Javascript Database

LokiJS is a javascript database that can be used in multiple scenarios.

Javascript is an easy to learn, versatile language, so database development in javascript can be easy and very productive. You may not want to retire your MySQL db just yet but here are some situationss in which you may find LokiJS to be the ideal solution:

  • Mobile applications - especially HTML5 based (Cordova, Phonegap, etc.)
  • Node.js embedded datastore for small-to-medium apps
  • Embedded in desktop application with Node Webkit

OOP, Functional, why choose between the two.

LokiJS leverage the power of javascript by being versatile.
If functional programming is your preferred style then you may want to create views to query your data.
If you prefer object literals in MongoDB shell style, you can use those.
If you prefer SQL, you're going to have to wait a bit, but it is coming.

Quick Start

Installation

LokiJS is available on npm and bower. Run:
npm install lokijs
or:
bower install lokijs

Usage

Create the database:
var db = new loki('Loki')

Create a collection:
var children = db.addCollection('children','NorseGod') Parameters are the collection name and the object Type which is an arbitrary string

Insert a document:
children.document({name:'Sleipnir', legs: 8})
children.document({name:'Jormungandr', legs: 0})
children.document({name:'Hel', legs: 2})

Retrieve documents:
children.find('name','Joe')
